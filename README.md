# Portfolio Project

## Overview
This project is a personal portfolio website built using React and GSAP. It showcases my projects, skills, and experience as a developer.

## Installation
To get a local copy up and running, follow these simple steps.

**Clone the repository:**
   ```bash
   git clone https://github.com/moe2008/portfolio.git
   cd portfolio
   ```
**Install NPM packages:**

```bash
npm install
```

**Start the development server:**

```bash
npm run dev
```
## Usage
To view the portfolio, open your browser and navigate to http://localhost:3000 (or the port specified in your environment).

## Features
Responsive Design: The website is fully responsive and works on all devices.
Smooth Animations: GSAP is used for creating smooth animations and transitions.
Project Showcase: Displays a list of my projects with descriptions and links.
Contact Form: A functional contact form for visitors to reach out to me.

## Technologies
**React:** JavaScript library for building user interfaces.
**GSAP:** GreenSock Animation Platform for creating animations.
**Vite:** Next Generation Frontend Tooling for fast and optimized builds.
**Tailwind CSS:** Utility-first CSS framework for styling.
